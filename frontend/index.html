How to Use the Assistant

1

Prepare your Pull Request

Open a Pull Request on GitHub and copy the full URL from your browser. It should look like: https://github.com/owner/repo/pull/123

2

Generate a GitHub Token

We need access to read your code. Create a Fine-grained Personal Access Token with these specific permissions:

  • Contents: Read-only
  • Pull requests: Read and Write
3

Initiate AI Analysis

Paste the PR URL and your Token into the dashboard. Optionally, specify a focus (e.g., "Check for memory leaks") and hit Analyze Pull Request.

The AI will process the diff, retrieve context using RAG, and post a categorized review comment directly on your GitHub PR.
